# Career Coaching Frameworks Reference

## GROWモデル — 拡張質問バンク

### Goal フェーズ
- 「このセッションが終わったとき、何を持ち帰りたいですか？」
- 「ゴールを一言で表すと？」
- 「そのゴールはあなたにとってなぜ重要ですか？」
- 「達成されたら何が変わっていますか？何を見て、聞いて、感じていますか？」
- 「そのゴールはコントロール可能なものですか？」

### Reality フェーズ
- 「現在どこにいますか？ゴールとのギャップは？」
- 「このテーマに関連して最近起きたことを話してください」
- 「誰がこの状況に影響していますか？」
- 「今何が機能していますか？何が機能していませんか？」
- 「これまでどんな行動を取りましたか？結果は？」

### Options フェーズ
- 「考えられる選択肢を全部出してください（良い悪いは関係なく）」
- 「もし時間・お金・スキルが無限にあれば何をしますか？」
- 「親友はどんなアドバイスをすると思いますか？」
- 「全く違うアプローチがあるとしたら？」

### Way Forward フェーズ（変化の確認）
※ 行動宣言を強要しない。情報の書き換わりを確認する。

- 「今日のセッションの最初と比べて、自分についての見え方は変わりましたか？」
- 「もし何かが変わっているとしたら、それは何ですか？」
- 「ここから自然に動きたいと思うことはありますか？」

## バランスホイール（ライフホイール）

### 概要
人生の各領域の満足度を可視化するコーチングツール。円を8つのスポークに分割し、各領域を0〜10で評価する。

**キャリアコーチングで最も重要な使い方**: 「仕事（やりがい）」と「収入（お金）」を**別のスポーク**として扱うこと。多くの人はこの2つを同一視しており、「やりたいことで食べていかなければならない」という思い込みを生む。この前提を早期に崩すことでキャリアの選択肢が広がる。

### 8領域
1. **仕事・キャリア** — やりがい、成長、貢献感
2. **収入・お金** — 経済的安定、豊かさ（仕事とは別軸）
3. **健康・体** — 体力、睡眠、運動
4. **人間関係** — パートナー、家族、友人
5. **学習・自己成長** — スキル、知識、探求
6. **遊び・趣味** — 楽しみ、創造、休息
7. **物理的環境** — 住環境、職場環境
8. **社会貢献・意義** — 使命感、社会との繋がり

### 導入タイミングと質問
セッション序盤、特にクライアントが「仕事とお金」を一緒に語っているときに活用する。

- 「仕事のやりがいと、収入の安定は、あなたの中で同じ問題ですか？別の問題ですか？」
- 「今の満足度をバランスホイールで見たとき、一番低いスポークはどこですか？」
- 「もし収入が確保されているなら、仕事に何を求めますか？」

## Will・Can・Must 詳細

### Will（やりたいこと） — 価値観と情熱
- 「子どもの頃、何に夢中でしたか？」
- 「時間を忘れて没頭できることは？」
- 「もし給与が同じなら何の仕事を選びますか？」
- 「どんな仕事をしているときに『これが私の仕事だ』と感じますか？」

### Can（できること） — スキルと強み
- 「これまでのキャリアで最も誇りに思う成果は？」
- 「人からよく頼まれることは何ですか？」
- 「自分では当たり前と思っているが、他の人には難しいことは？」

スキル棚卸しの軸:
1. 業務スキル
2. ヒューマンスキル
3. 専門知識
4. 言語スキル

### Must（求められること） — 市場の需要
- 「今の業界・職種の将来性をどう見ていますか？」
- 「あなたのスキルの市場価値は？」
- 「どんな組織・環境があなたを必要としていますか？」

## キャリアアンカー（Edgar Schein）詳細

### 1. 専門技術的能力
- 特徴: 専門的エキスパートであることに誇り
- 質問: 「何かのプロになることが最も大切ですか？」

### 2. 全般管理能力
- 特徴: 組織全体を動かす役割に満足感
- 質問: 「人や組織を率いることに充実感を感じますか？」

### 3. 自律・独立
- 特徴: 自分のペースと方法で働くことが最優先
- 質問: 「自分のルールで働けることが最も大切ですか？」

### 4. 保障・安定
- 特徴: 長期的な雇用保障と生活の安定を優先
- 質問: 「安定した雇用環境が何より大切ですか？」

### 5. 起業家的創造性
- 特徴: 自ら事業を作ること、リスクを取ること
- 質問: 「自分でビジネスを立ち上げることを夢見ますか？」

### 6. 奉仕・社会貢献
- 特徴: 社会や他者への貢献が仕事の意味
- 質問: 「世界をより良くすることが仕事の中心ですか？」

### 7. 純粋な挑戦
- 特徴: 困難な問題を解くこと自体が目的
- 質問: 「不可能に見える課題に挑むことが好きですか？」

### 8. 生活様式
- 特徴: 仕事とプライベートの両立が最優先
- 質問: 「仕事は人生の一部であり、生活全体のバランスが大切ですか？」

## 制限的信念へのアプローチ

| 制限的信念 | リフレーミング質問 |
|-----------|-----------------|
| 「私には経験が足りない」 | 「今あるスキルで何ができますか？」 |
| 「転職は失敗だ」 | 「転職した人の多くが良かったと感じています。なぜだと思いますか？」 |
| 「40代の転職は難しい」 | 「あなたの経験をほしい会社はどこですか？」 |
| 「安定を捨てるのは怖い」 | 「現状維持のリスクとは何ですか？」 |
| 「自分には強みがない」 | 「過去に誰かから感謝されたことは何ですか？」 |

## 価値観の明確化

仕事に関する価値観リスト（トップ10→3つに絞る）:
- 自律性 / 裁量権
- 安定性 / 保障
- 創造性 / 革新
- 専門性 / 深掘り
- リーダーシップ / 影響力
- チームワーク / 協力
- 社会貢献 / 意義
- 報酬 / 経済的成功
- 学習 / 成長
- ワークライフバランス
- 挑戦 / 達成感
- 認知 / 評価
- 多様性 / 変化
- 誠実さ / 倫理
- 顧客・社会への価値提供

## 変化の観測（次セッション冒頭）

宣言の履行確認ではなく、自然な変化を観測する:

- 「前回から何が変わりましたか？」
- 「前回のセッションで気づいたことが、その後どう影響しましたか？」
- 「何か自然に動いていたことはありましたか？」

**哲学**: 情報が書き換わった人間は、宣言しなくても動く。
