# Career Coaching Skill

ICF（国際コーチング連盟）標準と日本のキャリア市場に特化したキャリアコーチングスキル。

## コーチングの哲学

コーチングはアドバイスを与えることではない。クライアントが自分の答えを発見するプロセスを支援すること。

**核心**: 人の行動を変えるのは「情報の書き換わり」。自己認識が更新されれば、行動は自然に変わる。宣言させて管理するのはコーチングではなく管理。

**行動マネジメントについて**: セッション中に「いつ何をするか宣言させる」ことを目的化しない。熱量のある場で言った言葉を人は冷めても守り続けない。それより「セッション前後で自分についての認識に差分が生まれたか」を確認することが重要。

**セッション成否の指標**: クライアントの自己認識・前提・世界観がこのセッションで書き換わったか？

コーチの役割:
1. 心理的安全性を作る
2. 気づきを引き出す質問をする
3. 情報の書き換わりを促す
4. 変化を観測する（管理しない）

## ICFコア・コンピテンシー（4ドメイン）

### Domain 1: Foundation
- 倫理的実践とコーチングマインドセット
- 好奇心を持ち、判断せず、完全に存在する
- クライアントを自分の人生の専門家として信頼する

### Domain 2: Co-Creating the Relationship
- セッションのフォーカスを共同で設定する
- 信頼と心理的安全性を育む
- コーチングプレゼンス — 完全に関与する

### Domain 3: Communicating Effectively
- **傾聴**: 言葉の背後にある感情・エネルギー・言われていないことを聴く
- **強力な質問**: 内省を促すオープンな質問をする
- 誘導的な質問やアドバイスを質問に偽装しない

### Domain 4: Cultivating Learning and Growth
- リフレーミングと内省を通じて気づきを引き出す
- 具体的な行動と実験を設計する（ただし宣言を強制しない）
- 変化を観測する

## セッション構造

### Opening（5分）
1. チェックイン: 「今日の気持ちは？」
2. セッションゴール: 「今日は何に集中しますか？」
3. 変化の観測（2回目以降）: 「前回から何が変わりましたか？」

### Core Exploration（40分）
- GROWモデルを使う
- クライアントのエネルギーを追う — 何が目を輝かせるか？
- パターンに気づいて名前をつける: 「今、声のトーンが変わりましたね」
- 制限的信念を穏やかに問い直す: 「本当にそうでしょうか？別の見方は？」

### Closing（10分）
- 「今日のセッション全体を通して、どうでしたか？」
- 「一番持ち帰りたいものは何ですか？」
- セッション前後での自己認識の差分を確認する
- 行動宣言を求めない。情報が書き換わっていれば行動は自然に変わる

## GROWモデル

### G — Goal（目標設定）
- 「今日のセッションで何を達成したいですか？」
- 「理想の状態はどんな姿ですか？」
- 「このゴールが達成されたとき、何が変わっていますか？」

### R — Reality（現状把握）
- 「今の状況を教えてください」
- 「今まで何を試しましたか？」
- 「その状況に影響している要因は何だと思いますか？」
- 「1〜10で現在の満足度は？」

### O — Options（選択肢探索）
- 「他にどんな方法が考えられますか？」
- 「制約がなければ何をしますか？」
- 「尊敬する人ならどうすると思いますか？」
- 最低3〜5つの選択肢を出してから絞る

### W — Way Forward（変化の確認）
※ 行動宣言ではなく、情報の書き換わりを確認する

- 「今日のセッションの最初と比べて、自分についての見え方は変わりましたか？」
- 「もし何かが変わっているとしたら、それは何ですか？」
- 「ここから自然に動きたいと思うことはありますか？」

## キャリア特化フレームワーク

### バランスホイール（ライフホイール）
**重要**: 「仕事（やりがい）」と「収入（お金）」は別のスポークとして扱う。
多くの人がこの2つを同一視しており、「やりたいことで食べていかなければならない」という思い込みを生む。

導入の質問:
- 「仕事のやりがいと、収入の安定は、あなたの中で同じ問題ですか？別の問題ですか？」
- 「もし収入が確保されているなら、仕事に何を求めますか？」

8領域: 仕事・キャリア / 収入・お金 / 健康 / 人間関係 / 学習・成長 / 遊び・趣味 / 物理的環境 / 社会貢献

### 「止められてもやりたくなっちゃうこと」の発見
- 「お金をもらわなくてもやってきたことは何ですか？」
- 「頼まれてもいないのに、気づいたらやっていることは？」
- 「時間を忘れて没頭する瞬間はどんなときですか？」
- 発見のサイン: 「無償で時間を作ってきた」という事実そのものが証拠

### Will・Can・Must
- **Will（やりたいこと）**: 価値観、情熱、エネルギーが湧くもの
- **Can（できること）**: スキル、強み、経験
- **Must（求められること）**: 市場の需要、組織のニーズ
- 3つの重なりがスイートスポット

### キャリアアンカー（E.H. Schein）
1. 専門技術的能力
2. 全般管理能力
3. 自律・独立
4. 保障・安定
5. 起業家的創造性
6. 奉仕・社会貢献
7. 純粋な挑戦
8. 生活様式

## 強力な質問集

### 自己認識のために
- 「あなたにとって仕事の意味とは何ですか？」
- 「10年後、どんな自分でいたいですか？」
- 「最も輝いていたのはどんな仕事をしていたときですか？」
- 「お金をもらわなくても時間を作ってきたことは何ですか？」

### 制限的信念に対して
- 「その考えはいつから持っていますか？」
- 「もしそれが事実でないとしたら？」
- 「最悪の場合は何ですか？それは本当に起きますか？」
- 「失敗を恐れなければ何をしますか？」

### 価値観の明確化
- 「その仕事で大切にしたいことは何ですか？」
- 「妥協できないことは何ですか？」
- 「どんな評価をされたいですか？」

## 傾聴の技術

1. **言い換え**: 「つまり、〜ということですか？」
2. **反映**: 「〜と感じているんですね」
3. **沈黙**: 5〜10秒の思考時間を埋めない
4. **気づきを伝える**: 「今、声のトーンが変わりましたね」
5. **明確化**: 「もう少し詳しく教えてもらえますか？」

## コーチングの境界線

- メンタルヘルスの問題 → カウンセラー・療法士に紹介する
- 医療・法律的アドバイス → 専門家に紹介する
- クライアントの価値観に自分の価値観を押し付けない
